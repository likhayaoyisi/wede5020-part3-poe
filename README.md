BrightFix Plumbing Services Website
BrightFix Plumbing Services is an informative, interactive, and SEO-optimized website for a fictional Cape Town plumbing company. Built to demonstrate industry best practices, the site offers clear navigation, mobile-friendly design, and dynamic forms for user engagement.

Website Overview & Page Functions
Below is what each major section/page contains and its core purpose:

1. Home (index.html)
Function: Welcomes visitors with a summary of BrightFix’s expertise, emergency contact info, and reasons to choose the business.

Features: Eye-catching hero image, CTA for booking, brief value proposition.

2. About Us (about.html)
Function: Shares the company history, mission, values, and what makes BrightFix unique.

Features: Team or company photo, descriptive text, helps build trust and authenticity for new visitors.

3. Services (services.html)
Function: Showcases the complete list of plumbing services offered (repairs, installations, maintenance).

Features: Service descriptions with relevant images, highlighting expertise and options.

4. Contact (contact.html)
Function: Provides all essential business contact information and a contact form for general enquiries or urgent help.

Features: 24/7 phone number, email, and a user-friendly contact form with JavaScript validation.

5. Enquiry (enquiry.html)
Function: Advanced form for users to inquire specifically about products, services, volunteering opportunities, or sponsorship.

Features: Custom dropdown for enquiry type, detailed input, contextual feedback on submission, validation for data quality.

6. CSS & JS
css/styles.css: Contains all site styling for responsiveness, layout, color, spacing, and accessibility.

js/scripts.js & inline JS: Handles client-side form validation and error display to enhance usability.

7. SEO & Search Engine Essentials
robots.txt: Guides search engines about which site sections to index.

sitemap.xml: Lists all accessible pages for faster and more complete search engine indexing.

Meta & Alt Tags: All pages use meta descriptions and alt attributes for better discoverability and accessibility.

Navigation & User Flow
Visitors start on the Home page, learning about BrightFix’s services.

Easy navigation allows moving between About, Services, Enquiry, and Contact.

Forms are available on Contact and Enquiry for communication and requests.

Users receive instant feedback when submitting forms, making interactions smooth and reliable.

Features Overview
Mobile-Friendly: Responsive design works on all devices.

SEO Best Practices: Clear titles, descriptive meta tags, alt text, robots.txt, and sitemap.xml.

Accessibility: Semantic HTML, proper labels, easy keyboard navigation.

Client-Side Validation: Forms prevent incorrect data, guiding users to complete requests effectively.

Dynamic Interactivity: Instant on-page feedback after submitting forms.

Easy Customization: Code organized by separate HTML, CSS, and JavaScript files for continued improvement.
